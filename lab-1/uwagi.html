Formularze HTML bez wsprawcia JavaScript - uwagi

Mimo dużych możliwości formularze HTML bez wsparcia JavaScript posiadają też wady i ograniczenia.

Możliwości formularzy HTML

Formularze HTML posiadają podstawową możliwość walidacji formularzy. Walidacja jest wbudowana w przeglądarkę i nie wymaga instalowania zewnętrznych bibliotek. Dostępne jest wiele typów kontrolek - od tekstu aż po załączniki czy wybierak kolorów.

Wady i ograniczenia

Takie formularze dobrze sprawdzą się w prostych zastosowaniach, jednak w bardziej zastosowaniach aplikacjach powinno zastosować się również JavaScript dla poprawy wygody użytkownika. Dużą wadą jest to, że walidacja odbywa się dopiero przy próbie wysłania formularza, a nie od razu np. po przejściu do następnej kontrolki.

Ograniczona jest również możliwość zmiany stylu komunikatu walidacji. Na tę chwilę nie da się tego zrobić.
